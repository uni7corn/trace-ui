反编译视图的交互布局

点击选择你偏好的布局方式

A

右侧分栏（类似 IDA）

反编译伪代码作为右侧面板，与左侧 trace 汇编表格并排显示。点击 trace 中的某行汇编，右侧自动定位到对应的伪代码行并高亮。点击伪代码某行，左侧 trace 也联动跳转。

B

底部 Tab 面板（扩展现有 TabPanel）

在现有的 Memory / Taint / Search 等标签旁新增一个 "Decompile" 标签页。点击 trace 某行时，底部显示该函数的反编译伪代码并高亮当前行。占用空间更小，不影响现有布局。

C

可浮出的独立面板（两者兼得）

默认作为底部 Tab 页，但可以像现有面板一样拖拽浮出为独立窗口。浮出后可以自由调整大小和位置，放在第二屏上也行。复用现有的 FloatingPanel 机制。
